## 贡献指南

如果你想为项目添加新功能：

1. 查看 [Issues](链接到项目 Issues) 了解当前进展
2. Fork 项目并创建新分支
3. 实现功能并添加测试
4. 提交 Pull Request
5. 等待代码审查

## 功能请求

如果你有新的功能建议，请：

1. 检查是否已有类似的功能请求
2. 创建新的 Issue，使用 "Feature Request" 标签
3. 详细描述功能需求和使用场景
4. 如果可能，提供设计草图或示例
